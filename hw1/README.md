Overview
In this assignment, I modify the provided DFS (Depth-First Search) maze traversal
visualization code to implement the A* (A-star) pathfinding algorithm instead. This assignment
helps me to understand how informed search algorithms like A* improve upon uninformed
searches such as DFS by incorporating heuristic information to guide the search process more
efficiently.

Objectives
1. Understand the principles of informed search and heuristic functions
2. Implement the A* algorithm for maze pathfinding
3. Analyze how heuristic-guided search differs from uninformed search strategies
4. Evaluate the efficiency of A* compared to DFS in terms of path optimality and search
cost
